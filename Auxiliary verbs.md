# Auxiliary verbs

## Être (to be)

### Present (Présent)

| Person        | Conjugation |
|---------------|-------------|
| Je            | suis        |
| Tu            | es          |
| Il/Elle/On    | est         |
| Nous          | sommes      |
| Vous          | êtes        |
| Ils/Elles     | sont        |

### Imperfect (Imparfait)

`ét-` + verb endings.

| Person        | Conjugation | Verb Ending |
|---------------|-------------|-------------|
| Je            | étais       | -ais        |
| Tu            | étais       | -ais        |
| Il/Elle/On    | était       | -ait        |
| Nous          | étions      | -ions       |
| Vous          | étiez       | -iez        |
| Ils/Elles     | étaient     | -aient      |

### Future Simple (Futur Simple)

The future stem (`ser-`) + verb endings.

| Person        | Conjugation | Verb Ending |
|---------------|-------------|-------------|
| Je            | serai       | -ai         |
| Tu            | seras       | -as         |
| Il/Elle/On    | sera        | -a          |
| Nous          | serons      | -ons        |
| Vous          | serez       | -ez         |
| Ils/Elles     | seront      | -ont        |
